# how to use

## Using subdirectory

1. clone this repo
2. add symbolic shortcut

````
ln -s dotfiles/.bash_aliases .bash_aliases
ln -s dotfiles/.gitconfig .gitconfig

````

## Using ~

````
cd ~
git init
git remote add origin https://github.com/foo/bar.git
git pull
````
